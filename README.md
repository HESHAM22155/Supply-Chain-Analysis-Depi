## 🎯 Strategic Objectives
* **Data Sourcing:** Acquired datasets from **Kaggle**.
* **Data Architecture:** Transformed raw data into a scalable **Star Schema**.
* **Database Engineering:** Implemented a robust SQL-based environment.
* **Business Intelligence:** Developed dynamic dashboards for tracking performance.

---

## ⚙️ Technical Data Pipeline

### 1️⃣ Phase 1: SQL Engineering (The Engine)
We prioritized building a solid database foundation first.
* **Tool:** SQL Server.
* **Process:** * Cleaned data using SQL queries.
    * Segregated data into **Fact** and **Dimension** tables.
    * Ensured data integrity and optimized relationships for BI tools.

### 2️⃣ Phase 2: Excel Analytics & Dashboarding
* **ETL:** Used Power Query for final structural validation.
* **Visualization:** Created a fully dynamic dashboard where users can interact with data without needing technical SQL knowledge.

### 3️⃣ Phase 3: Future Roadmap 🚀
* **Python:** Exploratory Data Analysis (EDA) & Correlation analysis.
* **Power BI:** Advanced DAX measures and cloud-based reporting.
